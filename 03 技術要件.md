## 採用技術（稼働スタック）

| 区分 | 採用技術 | 備考 |
|—|—|—|
| 言語 | Python 3.11+ | 仮想環境を使用 |
| APIサーバ | FastAPI | 中央リレー（localhost） |
| リアルタイム通信 | WebSocket | UI/状態イベント |
| LLM | Ollama（最新） | 稼働時唯一のLLM（R-11） |
| ベクトルDB | Qdrant または Chroma | ローカル。どちらか一方をPHASE1で確定 |
| 知識ベース | Obsidian（Vault / Markdown） | RAG情報源（R-7） |
| STT（音声認識） | Whisper.cpp | ローカル |
| TTS（音声合成） | ローカルTTS（無料スタック優先） | 具体実装はPHASE6で確定 |
| 顔認証 | DeepFace / face_recognition | ローカル。AWS Rekognitionは使用しない |
| 感情分析 | FER | ローカル・提案のみ |
| フロントエンド | React | localhost API / WebSocket のみ |
| ブラウザ補助 | Chrome拡張 | キャラ表示補助のみ |
| 会議オーケストレーション | GAS | 補助保存用途のみ（R-4b） |

- 【禁止技術】クラウドLLM（稼働時）/ AWS Rekognition / 3D・AR・VRM・Unity（R-12）/ dify

## 外部サービス境界（Local First の整理）

> 関係を明確化する。原則と例外を混同しないこと。

### 原則（ローカル必須・外部禁止）

- AI処理：ローカルのみ（R-4a）
- LLM：Ollama限定（R-11）
- RAG：Obsidian限定（R-7）
- 外部API：禁止（R-7）
- 稼働時の外部送信：禁止（R-4）

### 例外（補助保存用途のみ）

- Google Docs：会議議事録の保存先
- Google Spreadsheet：会議ログ・URL記録
- GAS：会議進行・音声I/O中継・ログ保存の中継役（意思決定はしない）
- 【成功条件】Googleサービスの通信が「会議ログ／議事録の補助保存」に限定され、それ以外の全処理がローカルで完結している
- 【禁止事項】GoogleサービスのLLM用途利用 / RAG情報源化 / 顔データ・プロファイル・機密の送信

## ディレクトリ構成（疎結合モジュール）

```
local-ai/
├── api/          # FastAPI 中央リレー・WebSocket
├── llm/          # Ollama 接続・システムプロンプト
├── knowledge/    # Obsidian読込・チャンク・埋め込み・RAG・raw→wiki整理
├── auth/         # 顔認証・セッション・local_profile
├── camera/       # インターバル撮影・離席検知
├── emotion/      # FER感情分析（提案のみ）
├── voice/        # Whisper.cpp(STT) / ローカルTTS
├── video/        # 2D動画演出・クロスフェード
├── meeting/      # GAS会議連携（補助保存）
├── state/        # 状態管理（インメモリ＋ローカル永続化）
└── frontend/     # React（認証ボタン・動画ステージ・検索UI）
```

- 【成功条件】各ディレクトリがAPI/WebSocket経由でのみ連携し、相互のコード依存をハードコードしていない（R-10）
- 【禁止事項】`3d/` `ar/` `vrm/` `unity/` ディレクトリの作成（R-12）/ UI依存構造

## Obsidian Vault 構成

```
Vault/
├── raw/        # 投入ドロップゾーン（メモ/テキスト/PDF/保存済みWeb記事ファイル）
├── wiki/       # 整理後の構造化Markdown出力先（相互リンク付き）
├── Memory/     # 「覚えて」で書き込む記憶ノート
├── Knowledge/  # 恒久ナレッジ
└── Daily/      # ライフログ
```

- 【成功条件】Vault内に機密（APIキー・パスワード・Cookie）が存在しない（R-6）
- 【禁止事項】Vaultへの機密保存

-----

## local_profile 仕様

- 保持項目：`fullname` / `preferred_name` / `face_embedding` / `local_settings` のみ
- 保存先：ローカルのみ（クラウド同期なし）
- 【禁止事項】上記4項目以外の保持 / クラウド同期

-----

## オフライン検証要件

- 【目的】ローカル完結を実証する
- 【入力】ネットワーク切断状態
- 【出力】例外（会議ログ補助保存）以外の全機能が動作する状態
- 【成功条件】ネット切断でチャット・RAG・音声・顔認証・感情分析・動画が動作する
- 【禁止事項】稼働必須機能のクラウド依存
