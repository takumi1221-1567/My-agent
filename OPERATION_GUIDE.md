# RET — 操作説明書（Operation Guide）

> AI執事「あいなす」の使い方ガイド。スマホ（PWA）でそのまま使えます。
> 本番URL: https://your-project.pages.dev

---

## 1. はじめかた

1. スマホ/PCのブラウザで本番URLを開く（ホーム画面に追加するとアプリのように起動）。
2. **マイクとカメラの許可**を求められたら許可。
3. **認証画面**:
   - 顔認証で主ユーザーと認識されれば自動で起動。
   - 顔が使えない場合は **お名前 ＋ パスワード** を入力して「認証」。
4. 起動後、画面右上の**マイクボタン**で会話を開始。

> 💡 マイクを一度タップすると音声が有効化され、以降は動画にも音が出ます。

---

## 2. 基本の話しかた

1. **右上のマイクボタンを1回押す**（赤くなったら聞き取り中）。
2. **話しかける**（話し終わると自動送信）。
3. あいなすが**声と文字**で答える。止めたい時はもう一度マイクを押す。

---

## 3. 音声コマンド一覧

| 言葉（例） | 動作 |
|---|---|
| （自由な質問・雑談） | あいなすが回答（Obsidian知識・記憶を参照） |
| 「〇〇を**覚えて**」 | 確定事実として記憶（例:「私の出身は青森と覚えて」） |
| 「**クリア**」 | 取扱説明（マニュアル）を表示 |
| 「**外出**」 | 外出ドライブ演出を開始 |
| 「**もういいよ**」 | 外出/検索を終了して帰宅 |
| 「**行くぞ**」 | 共闘/単独シーケンスを開始 |
| 「**検索**」 | Web検索シーケンスを開始（パスワード要） |

> 認識ゆれに対応: 例「行くぞ/いくぞ」「もういいよ/もういい」「検索/しらべて」等も可。

---

## 4. 記憶させる（「覚えて」）

- 「〇〇を覚えて」と言うと、その内容を**確定事実**として記憶します。
- 記憶はクラウドに即保存され、以後の回答に反映。Mac（Obsidian）にも順次保存されます。
- 個人の事実（名前・出身・予定など）は、**覚えさせた内容だけ**が回答に使われます（推測しません）。

---

## 5. 外出ドライブ

1. 「**外出**」と言う → 外出→ドア→発進→後部座席（音声付き）。
2. 走行中は5秒ごとに雑談。話しかければ会話に応じます。
3. 「**もういいよ**」と言う → 停車→帰宅して通常モードへ。

---

## 6. 共闘／単独（「行くぞ」）

1. 「**行くぞ**」と言う → 「一緒にいかがですか？」と聞かれる。
2. **「はい」=共闘** / **「いいえ」=単独** で分岐。
3. バイク→（相乗り or ソロ）で走行。走行中の**独り言に返事すると会話**できます。
4. 「**着いたな**（共闘）/ **着いたぞ**（単独）」で到着 → バックルームへ。
5. 「**帰ろう**」で帰宅。

---

## 7. Web検索（パスワード認証付き）

1. 「**検索**」と言う → 「パスワードをどうぞ」。
2. **パスワード「214200」**（ツーワンフォーツーゼロゼロ）を音声で言う。
   - 数字・全角・英語読み・日本語読みのいずれでもOK。
3. 正答 → 研究室→パソコンの演出 → 「**何をお調べしますか？**」→ 検索ワードを言う。
4. 結果を読み上げ → 「**保存しますか？**」
   - 「はい」= Obsidianの `raw/` に保存（Mac起動中は数分以内、未起動なら次回起動時）。
   - 「いいえ」= 「検索を続けますか？」
5. 「**もういいよ**」で施錠→帰宅して終了。

> ⚠️ パスワードを間違えると、演出として「発狂大佐」のセリフが倍速で流れます。正しいパスワードを言えば検索に戻ります。

---

## 8. 予定（カレンダー）

- Google カレンダー連携を設定しておくと、起動時に**本日以降の予定を読み上げ**ます。
- 「今日の予定は？」等の質問にも予定を参照して回答します。
- ※連携用のGAS（Apps Script）作成は別途セットアップが必要です。

## 9. Discordで使う（EAST会議）

- ボットがいるDiscordチャンネルで**普通に発言**するだけ。
- 専門の異なる複数AIエージェント＋あいなすが、関係する時に自動で返答（最大3ラウンド）。
- 「**AI活用**」「ニュース」「自動化」などのキーワードを含むと、あいなすが要約で答えます。
- あなたが招待した友人の発言にも反応します。

## 10. こまったとき（FAQ）

| 症状 | 対処 |
|---|---|
| 「もう一度試して」と出る | マイクをもう一度押す |
| 音が出ない | 端末の音量・マナーモードを確認。一度マイクをタップして音声を有効化 |
| マイクが使えない | ブラウザのマイク許可をON |
| 認証で弾かれる | 名前は主ユーザー名、パスワードは正しい固定値を入力 |
| 検索が「見つからない」 | 別の言い回しで再検索（一般的な知識はWikipedia補完で回答） |
| 知識の質問に答えない | カタカナ語は英語名でも可。Obsidianに該当ノートがあるか確認 |

## 11. 運用・デプロイ（管理者向け）

### 本番デプロイ（Cloudflare Pages）
```
cd /path/to/RET
npx wrangler pages deploy public --project-name=ret --commit-dirty=true
```
- JS/CSSを変更したら `index.html` の参照クエリ `?v=N` を更新（キャッシュバスト）。

### ローカルAI（AINAS / Mac・任意）
- FastAPI（uvicorn :8000）。Obsidian Vault（iCloud）を読み書き・D1へ同期・KVから逆同期。
- 起動/再起動（launchd）:
  ```
  launchctl kickstart -k gui/$(id -u)/com.ainas.server
  ```
- 環境変数 `CF_SYNC_URL` / `CF_SYNC_TOKEN` が必要（Obsidian↔クラウド同期用）。

### Discordボット（EAST・任意）
- `discord_bot.py`（launchd `com.east.discord-bot`）。RETの `/api/chat` を会議参加者として呼ぶ。
  ```
  launchctl kickstart -k gui/$(id -u)/com.east.discord-bot
  ```

### Mac ON/OFF と機能

| 機能 | Mac OFFでも動く？ |
|---|---|
| 会話・質問・Discord回答・Obsidian知識参照 | ✅（クラウド70B + D1ミラー） |
| 「覚えて」記憶の反映 | ✅ 即時 |
| 検索結果/記憶の **Obsidian実ファイル書き込み** | ⚠️ Macが要る（ON=最大約2分 / OFF=次回起動時） |
